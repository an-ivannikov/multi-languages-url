Class: MultiLanguageURL

MultiLanguageURL(options)

new MultiLanguageURL(options)

The class of the multilanguage page location url. Extracts from `window.location.pathname` one of the list of valid languages ​​and insertes it at the beginning of the incoming location url. Forms the base of the path `this.rootLanguagePath` for subsequent substitution to the incoming location url.
